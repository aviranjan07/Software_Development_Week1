Array Operations (Core CRUD Thinking)

Arrays are one of the most important data structures in programming.
Almost every software system uses array-like structures.
In real-world software, arrays follow CRUD operations:
C → Create / Insert
R → Read / Access
U → Update
D → Delete
This is the same logic used in databases, APIs, and applications.
Example:
Operation
Example
Insert
Add a new user
Read
View user details
Update
Change email
Delete
Remove account

Arrays teach this core thinking.

1️⃣ Array Basics
What is an Array?
An array is a collection of elements stored in continuous memory locations.
Example:
Index:   0   1   2   3   4
Array:  [10, 20, 30, 40, 50]

Important points:
Index starts from 0
Fixed size (in many languages)
All elements usually same type
Example (Java):
int[] numbers = {10, 20, 30, 40, 50};


2️⃣ Access Element (READ Operation)
Access by Index
To get a value from the array:
int value = numbers[2];
System.out.println(value);

Output
30

Explanation
numbers[0] → 10
numbers[1] → 20
numbers[2] → 30

Real-World Example
A student system:
students[0] = "Rahul"
students[1] = "Amit"
students[2] = "Riya"

To get the second student:
students[1]


3️⃣ Insert Operation
Arrays have fixed size, so insertion often means:
placing a value in an empty index
or shifting elements
Example
Before
[10, 20, 30, 40]

Insert 25 at index 2

Shift elements right.
After
[10, 20, 25, 30, 40]

Java Example
int[] arr = new int[5];

arr[0] = 10;
arr[1] = 20;
arr[2] = 30;
arr[3] = 40;

arr[2] = 25;


4️⃣ Update Operation
Update means modify an existing element.
Example
Before
[10, 20, 30, 40]

Update index 1 → 200

After
[10, 200, 30, 40]

Java Example
arr[1] = 200;

This replaces the old value.

5️⃣ Delete Operation
Arrays cannot shrink easily, so deletion means:
shifting elements left
or replacing with default value
Example
Before
[10, 20, 30, 40, 50]

Delete index 2

Shift left
After
[10, 20, 40, 50]

Java Example
for(int i = 2; i < arr.length - 1; i++){
    arr[i] = arr[i+1];
}


6️⃣ Complete CRUD Example Program
public class ArrayCRUD {

    public static void main(String[] args) {

        int[] arr = {10,20,30,40,50};

        // READ
        System.out.println("Element at index 2: " + arr[2]);

        // UPDATE
        arr[1] = 200;

        // INSERT (simple replace)
        arr[3] = 35;

        // DELETE (shift left from index 2)
        for(int i=2;i<arr.length-1;i++){
            arr[i] = arr[i+1];
        }

        // Print array
        for(int i=0;i<arr.length;i++){
            System.out.print(arr[i] + " ");
        }

    }
}

Output example:
10 200 40 50 50


7️⃣ Time Complexity (Important for Interviews)
Operation
Time Complexity
Access
O(1)
Update
O(1)
Insert
O(n)
Delete
O(n)

Why?
Because elements must shift.
