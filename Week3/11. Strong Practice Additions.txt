1️⃣ ✅ To-Do List Manager
👉 (Array + CRUD + Traversal)
💡 Concept Used
Array operations
push / splice
traversal

🧪 Basic Version
let tasks = [];

// Add task
function addTask(task){
    tasks.push(task);
}

// Remove task
function removeTask(index){
    tasks.splice(index, 1);
}

// Show tasks
function showTasks(){
    for(let i = 0; i < tasks.length; i++){
        console.log(i + ": " + tasks[i]);
    }
}


🧠 Upgrade Idea
Mark task as completed
Store as object:
{task: "Study", done: false}

3️⃣ ✅ Word Frequency Counter
👉 (String + Object + Counting Logic)

🧪 Example
let text = "apple banana apple mango banana apple";

let words = text.split(" ");

let freq = {};

for(let i = 0; i < words.length; i++){
    let word = words[i];

    if(freq[word]){
        freq[word]++;
    } else {
        freq[word] = 1;
    }
}

console.log(freq);


🧠 Output
apple: 3
banana: 2
mango: 1


🧠 Upgrade Idea
Ignore case (toLowerCase())
Remove punctuation

4️⃣ ✅ Find Highest & Lowest Marks
👉 (Traversal + Condition)

🧪 Example
let marks = [45, 67, 89, 23, 90];

let max = marks[0];
let min = marks[0];

for(let i = 1; i < marks.length; i++){
    if(marks[i] > max){
        max = marks[i];
    }
    if(marks[i] < min){
        min = marks[i];
    }
}

console.log("Max:", max);
console.log("Min:", min);


🧠 Upgrade Idea
Use array of objects (students)
Find topper name

5️⃣ ✅ Filter Expensive Products
👉 (Filter + Map + Real App Logic)

🧪 Example
let products = [
  {name: "Laptop", price: 50000},
  {name: "Phone", price: 20000},
  {name: "Mouse", price: 500}
];

let expensive = products.filter(p => p.price > 10000);

console.log(expensive);


🧠 Upgrade Idea
👉 Get only names:
let names = products
    .filter(p => p.price > 10000)
    .map(p => p.name);


🔥 MASTER LEVEL COMBINATION
👉 Combine everything:
products
  .filter(p => p.price > 10000)
  .map(p => p.name)
  .join(", ");
