8. Searching Techniques
👉 This is the first step toward algorithms & problem solving

🧠 Big Idea
👉 Searching = finding data inside a collection
Real-world examples:
🔍 Search user by email
🛒 Find product by ID
🎓 Find student by marks
📱 Check if username exists

1️⃣ Linear Search (Basic & Most Important)
👉 Concept
Check each element one by one

Example
let arr = [10, 20, 30, 40];

let target = 30;

for(let i = 0; i < arr.length; i++){
    if(arr[i] === target){
        console.log("Found at index:", i);
    }
}


🧠 How it works
Check 10 ❌  
Check 20 ❌  
Check 30 ✅ → Found


⏱ Time Complexity
👉 O(n) (worst case)

🧠 Real Use
Small datasets
Simple apps
Base logic for all searching

2️⃣ Find by Condition (Smart Search)
👉 Concept
Find element using a condition

Example
let users = [
  {name: "Rahul", age: 25},
  {name: "Amit", age: 30}
];

let result = users.find(user => user.age > 26);

console.log(result);

Output:
{name: "Amit", age: 30}


🧠 Real Use
Find user above age
Find product with price > 1000
Filter data dynamically

3️⃣ Existence Check (True / False)
👉 Concept
Check if element exists or not

Example 1: includes()
let arr = [10, 20, 30];

console.log(arr.includes(20));

Output:
true


Example 2: some()
let users = [
  {name: "Rahul"},
  {name: "Amit"}
];

let exists = users.some(user => user.name === "Amit");

console.log(exists);

Output:
true


🧠 Real Use
Check login user exists
Validate duplicate entries
Verify data before insert

⚠️ Common Mistakes
❌ Forgetting return in find
users.find(user => { user.age > 25 }) ❌

✅ Correct
users.find(user => user.age > 25)


❌ Using includes for objects
users.includes({name: "Rahul"}) ❌

👉 Objects need condition-based search
