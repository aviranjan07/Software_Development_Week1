2. Array Traversal Patterns
👉 This is where you become strong in loops + thinking like a developer


Traversal means visiting each element of an array one by one.
Think like this:
Array = Data
Traversal = How you process that data
This is used everywhere in real software:
Showing products in e-commerce
Filtering users
Processing API data
Calculating totals

1️⃣ Forward Traversal (Most Common)
👉 Concept
Start from index 0 → last index
Java Example
int[] arr = {10, 20, 30, 40, 50};

for(int i = 0; i < arr.length; i++){
    System.out.println(arr[i]);
}

Output
10
20
30
40
50

🧠 Real-World Use Case
Displaying items in UI
Printing reports
Sending data sequentially

2️⃣ Reverse Traversal
👉 Concept
Start from last index → 0
Java Example
for(int i = arr.length - 1; i >= 0; i--){
    System.out.println(arr[i]);
}

Output
50
40
30
20
10

🧠 Real-World Use Case
Undo operations
Stack-like behavior
Reverse display (chat messages, logs)

3️⃣ Conditional Traversal (Smart Filtering)
👉 Concept
Traverse + apply condition
Example: Print even numbers
for(int i = 0; i < arr.length; i++){
    if(arr[i] % 2 == 0){
        System.out.println(arr[i]);
    }
}

Output
10
20
30
40
50


More Examples
🔹 Find maximum value
int max = arr[0];

for(int i = 1; i < arr.length; i++){
    if(arr[i] > max){
        max = arr[i];
    }
}
System.out.println("Max: " + max);


🔹 Count odd numbers
int count = 0;

for(int i = 0; i < arr.length; i++){
    if(arr[i] % 2 != 0){
        count++;
    }
}
System.out.println("Odd count: " + count);

⚠️ Common Mistakes
❌ Wrong loop condition
i <= arr.length   // ❌ ERROR

✅ Correct
i < arr.length


❌ Infinite loop
for(int i = 0; i >= 0; i++)  // ❌ never ends


❌ Skipping elements accidentally
