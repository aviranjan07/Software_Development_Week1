3. Common Array Methods (Language Independent Thinking)


These are built-in helper methods that make array work easy and fast.
👉 Even though syntax changes (Java, Python, JavaScript),
👉 Concept is always SAME.

🧠 Big Idea (Very Important)
All array methods are just shortcuts for CRUD + Traversal
Method
Meaning
push / append
Insert element
pop
Delete last element
slice
Copy part of array
splice
Modify original array
length
Get size


1️⃣ push / append (INSERT)
👉 Concept
Add element at the end

🔹 JavaScript Example
let arr = [10, 20, 30];

arr.push(40);

console.log(arr);

Output:
[10, 20, 30, 40]


🔹 Python Example
arr = [10, 20, 30]
arr.append(40)


🧠 Real-World Use
Add new user
Add cart item
Add message in chat

2️⃣ pop (DELETE)
👉 Concept
Remove last element

JavaScript Example
let arr = [10, 20, 30];

arr.pop();

console.log(arr);

Output:
[10, 20]


🧠 Real-World Use
Remove last notification
Undo last action
Stack operations

3️⃣ slice (COPY / READ)
👉 Concept
Get a portion of array (without changing original)

JavaScript Example
let arr = [10, 20, 30, 40, 50];

let newArr = arr.slice(1, 4);

console.log(newArr);

Output:
[20, 30, 40]


🧠 Important Rule
slice(start, end)
end is NOT included


🧠 Real-World Use
Pagination (show 10 items)
Extract data
UI rendering

4️⃣ splice (MODIFY ORIGINAL ARRAY)
👉 Concept
Insert / delete / replace elements

Example 1: Delete
let arr = [10, 20, 30, 40];

arr.splice(1, 2);

Output:
[10, 40]


Example 2: Insert
arr.splice(1, 0, 25);


Example 3: Replace
arr.splice(1, 1, 99);


🧠 Real-World Use
Edit database-like data
Remove user
Update list dynamically

5️⃣ length (SIZE)
👉 Concept
Get number of elements

Example
let arr = [10, 20, 30];

console.log(arr.length);

Output:
3


🧠 Real-World Use
Loop control
Validation
Data count

❌ Modifying array unintentionally
