5. Array of Objects (🔥 Very Important)
👉 This is where programming becomes REAL software development

🧠 Big Idea
👉 Combine Array + Object
Array → stores multiple items
Object → stores structured data
✅ Together:
[
  {name: "Rahul", age: 25},
  {name: "Amit", age: 30}
]

👉 This is how 90% of real-world data looks

🔥 Real Examples
👨‍🎓 Student List
let students = [
  {name: "Rahul", marks: 80},
  {name: "Amit", marks: 75},
  {name: "Riya", marks: 90}
];


🛒 Product Catalog
let products = [
  {id: 1, name: "Laptop", price: 50000},
  {id: 2, name: "Phone", price: 20000}
];


🏢 Employee Database
let employees = [
  {id: 101, name: "John", role: "Manager"},
  {id: 102, name: "Sara", role: "Developer"}
];


1️⃣ Access Data (READ)
👉 Get first student name
console.log(students[0].name);


👉 Loop through all
for(let i = 0; i < students.length; i++){
    console.log(students[i].name);
}


2️⃣ Insert (CREATE)
👉 Add new student
students.push({name: "Neha", marks: 85});


3️⃣ Update
👉 Update marks of Rahul
students[0].marks = 95;


4️⃣ Delete
👉 Remove a student
students.splice(1, 1);


🔥 Most Important Skill → Traversal + Condition
👉 Find topper
let topper = students[0];

for(let i = 1; i < students.length; i++){
    if(students[i].marks > topper.marks){
        topper = students[i];
    }
}

console.log(topper.name);


👉 Filter students above 80
for(let i = 0; i < students.length; i++){
    if(students[i].marks > 80){
        console.log(students[i].name);
    }
}


🌍 Real API Response (Exactly Same Structure)
