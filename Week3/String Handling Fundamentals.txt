7. String Handling Fundamentals
👉 This is used everywhere: validation, forms, APIs, search, parsing

🧠 Big Idea
👉 String = Text Data
Examples:
User name
Email
Password
Messages
API responses
👉 If you can handle strings → you can build real apps

1️⃣ length (SIZE)
👉 Concept
Find number of characters

Example
let str = "Hello";

console.log(str.length);

Output:
5


🧠 Real Use
Password validation
Input checking
Data limits

2️⃣ substring / slice (EXTRACT)
👉 Concept
Get part of a string

Example
let str = "Hello World";

console.log(str.slice(0, 5));

Output:
Hello


🧠 Rule
slice(start, end)
end is NOT included


🧠 Real Use
Extract username from email
Format display text
Show preview text

3️⃣ split (STRING → ARRAY)
👉 Concept
Break string into parts

Example
let str = "apple,banana,mango";

let arr = str.split(",");

console.log(arr);

Output:
["apple", "banana", "mango"]


🧠 Real Use
CSV data processing
Form input parsing
Tags handling

4️⃣ join (ARRAY → STRING)
👉 Concept
Convert array → string

Example
let arr = ["apple", "banana", "mango"];

let str = arr.join("-");

console.log(str);

Output:
apple-banana-mango


🧠 Real Use
Generate URLs
Format output
Combine words

5️⃣ Case Conversion
👉 Concept
Change uppercase / lowercase

Example
let str = "hello";

console.log(str.toUpperCase());
console.log(str.toLowerCase());

Output:
HELLO
hello


🧠 Real Use
Case-insensitive search
Login validation
Data normalization

6️⃣ Trimming (REMOVE SPACES)
👉 Concept
Remove extra spaces

Example
let str = "   hello   ";

console.log(str.trim());

Output:
hello


🧠 Real Use
Form input cleaning
Avoid errors in login
Data processing

🔥 Real Example (Form Handling)
let email = "   user@gmail.com   ";

email = email.trim().toLowerCase();

console.log(email);

👉 Clean + standardized data

⚠️ Common Mistakes
❌ Forgetting trim
" user@gmail.com " !== "user@gmail.com"


❌ Wrong slice index

❌ Case mismatch
"HELLO" !== "hello"
