What Programming Actually Is
1. The Simplest Meaning
Programming means giving instructions to a computer so it knows what to do.
Just like:
A teacher gives instructions to students
A chef follows instructions in a recipe
A driver follows road signs
Similarly, a computer follows instructions written by humans.
These instructions are called programs.
So,
Programming = Writing instructions for computers.

2. Why Computers Need Programming
Computers are very powerful but very dumb.
They cannot think.
They only do what they are told to do.
Example:
If you open a calculator and press 2 + 3
The computer doesn't "know math".
Instead it follows a program written by programmers that says:
Take first number
Take second number
Add them
Show the result

Without programming, a computer would just be a box with electricity.
Programming gives life to computers.

3. Real Life Example
Imagine a robot in your kitchen.
If you say:
"Make tea"
The robot will not understand.
You must give step-by-step instructions.
Example:
Boil water
Add tea leaves
Add milk
Add sugar
Stir
Pour into cup
That step-by-step instruction is exactly what programming is.
Computers need extremely clear instructions.

4. Programming Languages
Computers actually understand only binary language:
0 and 1

Example:
101010101010

Humans cannot easily write programs like that.
So we created programming languages.
These languages help humans talk to computers.
Examples of programming languages:
C
C++
Python
Java
JavaScript
Go
Rust
Each language is just a different way to give instructions.
Example in Python:
print("Hello")

This tells the computer:
Show the word "Hello" on the screen.

5. What Happens Inside the Computer
When you write a program:
Step 1 → You write code
Step 2 → The computer translates the code
Step 3 → It becomes machine language (0 and 1)
Step 4 → The CPU executes the instructions
This translation is done by:
Compiler
Interpreter

6. What Programmers Actually Do
Programmers solve problems using code.
Example problems:
Build a website

Make a game
Control robots
Analyze data

Run bank systems
Programming is basically problem solving with computers.

3. Taking Input in Java
Java uses a class called:
Scanner

The Scanner class reads user input from keyboard.

Step 1: Import Scanner
At the top of the program:
import java.util.Scanner;

This tells Java:
I want to use the Scanner tool.

Step 2: Create Scanner Object
Scanner sc = new Scanner(System.in);

Meaning:
Scanner → class
sc → object name
System.in → keyboard input

Step 3: Read Input
Example:
int age = sc.nextInt();

This reads an integer value from the user.

Full Example Program
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("Enter your age:");
        int age = sc.nextInt();
        System.out.println("Your age is " + age);
    }
}


How the Program Works
Step 1
Program asks:
Enter your age:

Step 2

20

Step 3
Program prints:
Your age is 20

2. if Statement
Simple Idea
The if statement checks a condition.
If the condition is true, the code runs.
If the condition is false, the code is skipped.

Real Life Example
Imagine a rule:
If it rains → take an umbrella

If it does not rain → do nothing.

Java Syntax
if(condition)
{
   code
}


Example
int age = 20;

if(age >= 18)
{
    System.out.println("You are an adult");
}

Output
You are an adult

Because 20 ≥ 18 is true.

3. if–else Statement

4. Example 2: Find Even or Odd Number
Logic
Take a number
Divide it by 2
If remainder = 0 → Even
Otherwise → Odd
Java Code
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        System.out.println("Enter a number:");
        int num = sc.nextInt();
        if(num % 2 == 0)
        {
            System.out.println("Even Number");
        }
        else
        {
            System.out.println("Odd Number");
        }

    }
}

Explanation:
%  → modulus operator

It finds the remainder.
Example:
10 % 2 = 0

So it is even.
