# 🔹 8. Basic Input Validation (Very Important for Real Applications)

**Input Validation** means:

> Checking whether the **user input is correct, safe, and acceptable** before the program processes it.

In real software, users can enter **wrong or unexpected data**, so programs must **validate input first**.

Without validation, programs may:

```text
crash
produce wrong results
create security problems
```

Examples of real systems needing validation:

```text
ATM withdrawal
login forms
online registration
payment systems
```

---

# 1️⃣ Why Input Validation is Important

Imagine a program asking for **age**.

User enters:

```text
-5
500
abc
```

These are **invalid inputs**.

The program must check:

```text
Is the input numeric?
Is it within valid range?
Is it empty?
```

This process is **input validation**.

---

# 2️⃣ Basic Input Validation Example

Java example:

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int age;

        System.out.println("Enter your age:");
        age = sc.nextInt();

        if(age >= 18){
            System.out.println("Eligible to vote");
        }
        else{
            System.out.println("Not eligible");
        }

    }
}
```

But this program **does not check invalid numbers**.

Better validation is needed.

---

# 3️⃣ Range Checking

Range checking ensures input lies within a **valid range**.

Example: Age must be between **1 and 120**.

Improved code:

```java
if(age >= 1 && age <= 120){
    System.out.println("Valid age");
}
else{
    System.out.println("Invalid age entered");
}
```

Range checking prevents:

```text
negative numbers
impossible values
incorrect data
```

---

# 4️⃣ Input Validation Using Loop

Often we repeat input until user enters correct value.

Example:

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        int age;

        do{

            System.out.println("Enter age (1-120):");
            age = sc.nextInt();

        }while(age < 1 || age > 120);

        System.out.println("Valid age entered");

    }
}
```

Logic:

```text
Ask input
Check condition
Repeat if invalid
```

This ensures **correct data before continuing**.

---

# 5️⃣ Error Messages

A good program tells the user **what went wrong**.

Bad program:

```text
Invalid
```

Better program:

```text
Error: Age must be between 1 and 120
```

Example:

```java
if(age < 1 || age > 120){
    System.out.println("Error: Enter age between 1 and 120");
}
```

Clear messages help users **correct their input**.

---

# 6️⃣ Real Example — ATM Withdrawal

Imagine ATM withdrawal.

Rules:

```text
Amount must be positive
Amount must be multiple of 100
Amount must be less than balance
```

Example:

```java
int balance = 5000;
int withdraw;

Scanner sc = new Scanner(System.in);

System.out.println("Enter withdrawal amount:");
withdraw = sc.nextInt();

if(withdraw <= 0){
    System.out.println("Invalid amount");
}
else if(withdraw > balance){
    System.out.println("Insufficient balance");
}
else{
    System.out.println("Withdrawal successful");
}
```

This is **basic input validation**.

# 7️⃣ Login Validation Example

Example login check:

```java
String username = "admin";
String password = "1234";

if(username.equals("admin") && password.equals("1234")){
    System.out.println("Login successful");
}
else{
    System.out.println("Invalid username or password");
}
```

Real systems also check:

```text
empty fields
invalid characters
minimum password length
```

# 8️⃣ Form Validation Example

Basic checks:

name not empty

if(password.length() < 8){

# 🔟 Developer Best Practices

Professional developers follow these rules:

```text
Validate input early
Show clear error messages
Repeat input if invalid
Never trust user input
```

A famous programming rule:

> **"Never trust user input."**

Because users may enter **anything**.

---

# Example of Full Validation

```java
Scanner sc = new Scanner(System.in);

int number;

do{

    System.out.println("Enter number between 1 and 10:");
    number = sc.nextInt();

    if(number < 1 || number > 10){
        System.out.println("Invalid input, try again.");
    }

}while(number < 1 || number > 10);

System.out.println("Valid number entered.");
```

This program **forces valid input**.
