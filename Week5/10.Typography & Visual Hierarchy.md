## 🧠 1. What is Typography?

Typography = **how text looks and feels**

👉 Includes:

* Fonts
* Sizes
* Spacing
* Alignment

---

## 🧠 2. What is Visual Hierarchy?

> 👀 Guiding user’s eyes to what matters first

Example:

* Big → Title
* Medium → Subtitle
* Small → Description

## 🔤 3. Font Pairing (VERY IMPORTANT)

👉 Use **2 fonts max**

---

### 🔥 Best Combinations

* Heading → Bold font
* Body → Simple readable font

Examples:

* Heading: **Poppins / Montserrat**
* Body: **Open Sans / Roboto**

---

### 🧱 Example

```css id="t1"
body {
  font-family: 'Open Sans', sans-serif;
}

h1, h2 {
  font-family: 'Poppins', sans-serif;
}
```

---

## 📏 4. Font Sizes & Scale

👉 Follow hierarchy:

```css id="t2"
h1 { font-size: 2.5rem; }
h2 { font-size: 2rem; }
p  { font-size: 1rem; }
```

---

### 🔥 Use `rem` (Responsive)

* Scales with device
* Better accessibility

---

## 🧠 5. Spacing (UNDERRATED POWER)

👉 Good spacing = clean UI

---

### 🔹 Line Height

```css id="t3"
p {
  line-height: 1.6;
}
```

👉 Improves readability

---

### 🔹 Letter Spacing

```css id="t4"
h1 {
  letter-spacing: 1px;
}
```

---

### 🔹 Margin Between Sections

```css id="t5"
section {
  margin-bottom: 40px;
}
```

---

## 👀 6. Readability Rules

✔ Keep line length ~50–75 characters
✔ Avoid very small fonts
✔ Use proper contrast
✔ Don’t overcrowd text

---

## 🎨 7. Color Contrast (CRITICAL)

👉 Text must be easy to read

---

### ❌ Bad

```css id="t6"
color: lightgray;
background: white;
```

---

### ✅ Good

```css id="t7"
color: #222;
background: #fff;
```

---

### 🔥 Accessibility Standard

* Follow WCAG contrast rules
* Tools like WebAIM Contrast Checker help test

---

## ⚠️ 10. Common Mistakes

❌ Too many fonts
❌ Low contrast text
❌ No spacing (crowded UI)
❌ Same size everywhere
❌ Using px instead of rem
