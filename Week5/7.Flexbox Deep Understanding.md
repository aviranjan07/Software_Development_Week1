👉 Everything in Flexbox depends on **axis**

### 🔹 Default (`flex-direction: row`)

* **Main Axis → Horizontal ➡️**
* **Cross Axis → Vertical ⬇️**

---

### 🔹 If you change direction:

```css
.container {
  flex-direction: column;
}
```

👉 Axes switch:

* Main Axis → Vertical
* Cross Axis → Horizontal

---

## 🧠 Rule to Remember (VERY IMPORTANT)

> 🎯 `justify-content` → controls **MAIN axis**
> 🎯 `align-items` → controls **CROSS axis**

---

## ⚙️ 2. justify-content (Main Axis Control)

```css
.container {
  justify-content: center;
}
```

### 🔥 Values:

* `flex-start` → left
* `center` → center
* `flex-end` → right
* `space-between` → equal gaps
* `space-around` → equal space around

---

## ⚙️ 3. align-items (Cross Axis Control)

```css
.container {
  align-items: center;
}
```

### 🔥 Values:

* `stretch` (default)
* `center`
* `flex-start`
* `flex-end`

---

## 🧠 4. gap (MODERN SPACING)

```css
.container {
  gap: 20px;
}
```

👉 Clean spacing between items
👉 Better than margin hacks ✅

---

## 🧠 5. flex-grow (Expansion Power)

```css
.item {
  flex-grow: 1;
}
```

👉 Items expand to fill space

---

### Example:

```css
.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }
```

👉 Item2 gets **double space**

---

## 🧠 6. flex-shrink (Shrink Control)

```css
.item {
  flex-shrink: 0;
}
```

👉 Prevents shrinking

---

## 🧠 7. flex-basis (Starting Size)

```css
.item {
  flex-basis: 200px;
}
```

👉 Initial size before grow/shrink

---

## 🔥 8. The Ultimate Shortcut

```css
.item {
  flex: 1;
}
```

👉 Means:

```css
flex-grow: 1;
flex-shrink: 1;
flex-basis: 0;
```

---

## 🏗️ 9. Real-World Layout Example (IMPORTANT)

### 👉 Center Everything Perfectly

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
```

👉 Perfect centering 🔥

---

### 👉 Equal Cards Layout

```css
.container {
  display: flex;
  gap: 20px;
}

.card {
  flex: 1;
}
```

## ⚠️ 10. Common Mistakes (BIG ONES)

❌ Confusing axes
❌ Using `align-items` for horizontal (wrong!)
❌ Not setting `display: flex`
❌ Using margins instead of `gap`
❌ Ignoring `flex-wrap`
